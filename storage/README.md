# 存储方案

> 本分类收录 Kubernetes 持久化存储、分布式文件系统相关技术文档

---

## 📚 技术列表

### etcd
高可用分布式键值存储,Kubernetes 控制平面的数据存储后端。

**适用场景**: Kubernetes 集群元数据存储、服务发现、配置管理

---

### Ceph
统一存储系统,提供对象存储、块存储和文件系统。

**适用场景**: 生产级分布式存储、私有云存储方案

---

### JuiceFS
云原生分布式文件系统,基于对象存储构建 POSIX 兼容文件系统。

**适用场景**: AI 训练数据存储、大数据分析、跨云数据共享

---

### GPFS (IBM Spectrum Scale)
高性能并行文件系统,适用于 HPC 和大数据场景。

**适用场景**: 超算中心、高性能计算、大规模数据分析
