# 硬件管理

> 本分类收录 Kubernetes 硬件资源管理相关技术文档

---

## 📚 技术列表

### NFD (Node Feature Discovery)
自动发现 Kubernetes 节点的硬件特性和能力。

**适用场景**: 异构集群调度、GPU 节点识别、硬件亲和性调度

---

### GPU Operator
NVIDIA GPU 资源管理,自动化 GPU 驱动、CUDA、容器运行时配置。

**适用场景**: AI/ML 工作负载、GPU 集群管理
