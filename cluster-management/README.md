# 集群管理

> 本分类收录 Kubernetes 集群部署、管理和多租户相关技术文档

---

## 📚 技术列表

### Kubespray
生产级 Kubernetes 集群部署工具,使用 Ansible 实现自动化部署。

**适用场景**: 裸金属、私有云、混合云 Kubernetes 集群部署

---

### Kamaji
多租户 Kubernetes 控制平面管理平台,支持共享工作节点的控制平面隔离。

**适用场景**: 多租户 SaaS 平台、边缘计算场景

---

### vCluster
虚拟 Kubernetes 集群,在单个物理集群中创建多个隔离的虚拟集群。

**适用场景**: 开发测试环境隔离、多团队共享集群
