# 镜像仓库

> 本分类收录容器镜像存储、分发和加速相关技术文档

---

## 📚 技术列表

### Harbor
企业级容器镜像仓库,提供安全扫描、镜像签名、复制等功能。

**适用场景**: 企业私有镜像仓库、镜像安全管理

---

### Spegel
基于 P2P 的容器镜像分发加速方案。

**适用场景**: 大规模集群镜像分发、降低镜像拉取时间

---

### Registry
Docker 官方镜像仓库实现。

**适用场景**: 轻量级私有镜像仓库、开发测试环境

---
